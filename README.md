# Frontend Developer Portfolio — portfolio-anti

An elegant, highly responsive personal portfolio website engineered to showcase professional frontend development skills, creative projects, and advanced technical capabilities. 

This project implements a modern, high-performance tech stack using a dynamic component-driven architecture, advanced motion mechanics, and an automated serverless email dispatch pipeline.

*🔗 Live Demo:* [https://shayannaqvi45.github.io/liquid-glass-portfolio/](https://shayannaqvi45.github.io/liquid-glass-portfolio/)

---

## 🚀 Features

- *Dynamic Architecture:* Features a modular component-driven interface injected dynamically through a root document framework.
- *Fluid Micro-interactions:* Powered by scroll-based motion physics and responsive interactive timeline animations.
- *Advanced SEO & Open Graph:* Fully configured search engine optimization meta-layers and Open Graph tags for rich social media preview visibility.
- *Automated Communication Pipeline:* An automated client-side inquiry system that intercepts contact form data and routes email transmissions directly to the recipient without requiring a dedicated backend server.
- *Standardized Visual Layouts:* Built with a uniform inline vector line iconography system for precise scalable asset delivery.
- *Ultra-Fast Static Analysis:* Enforces clean codebase execution constraints via high-speed, Rust-based static code analysis rules.

---

## 🛠️ Tech Stack

### Core Architecture
- *React 19 & React-DOM:* For managing declarative UI tree state mutations, component lifecycle scopes, and reactive user state bindings.
- *Vite 8:* Local bundling engine providing lightning-fast Hot Module Replacement (HMR) compilation layers.

### Styling & Visual Motion
- *Tailwind CSS v4 & @tailwindcss/vite:* Utilizes modern utility class layers integrated directly through a high-performance build-time design engine.
- *Framer Motion:* Handles interactive physics, entry/exit visual states, and complex element translation grids.
- *Lucide React:* Implements a predictable, standardized toolkit of vector line icon systems.

### Operations & Utilities
- *EmailJS-Com:* Safely intercepts form verification triggers to dispatch communications securely directly to target email endpoints.
- *Oxlint:* An ultra-high-speed, Rust-backed linter executing complex static analysis parameters to guarantee clean production compilations.
- *HTML5:* Forms the structural, semantic backbone of the layout alongside baseline metadata injections.
